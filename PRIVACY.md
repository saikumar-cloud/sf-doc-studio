# Privacy Policy - SF Doc Studio

Last updated: July 2026

We do NOT collect any data. All data stays on your local machine.

What is sent externally:
- Salesforce API: queries sent to YOUR org only
- Anthropic API: metadata sent for AI analysis

Data stored locally in Chrome only:
- Salesforce access token
- Salesforce instance URL
- Anthropic API key

To delete: Settings -> Disconnect and Clear All Data
